Insapp

Insapp is an application promoting associations within INSA Rennes.

insapp.fr

Repository (Android version) | Repository (server)

What's Insapp ?

Insapp is a 2016 project. This application helps associations from INSA Rennes to easily communicate with students. Associations have access to a web app to post some news just as Instagram or post events. Then, our users receive a push notification and get access to the content.

What I did

I joined the team in October (there was then 2 people only). We decided to rewrite the Android application, the first version being unstable and not material design-friendly. I used Retrofit as a HTTP client, Firebase Cloud Messaging to send push notifications and Fabric to monitor and understand crashes. I also made some experiments with the Kotlin language. The Android repository is hosted on Github.

I am now in charge of the maintenance of the project. I learned Golang and MongoDB so I could make changes to the server code, including regarding the REST API. The whole server is dockerized.

This project has taught me a lot in the sense that it is technically transversal, but also because it involves communicating with students and taking team decisions.

Usage

Insapp is available on the App Store and Play Store. Today, we count more than 800 active users: about 200 hundred downloads on iOS and 600 on Android. We worked with the school administration so they can also use the application. It's a large project in our school. Now, we try to pass our project to other students so they can innovate again and again on Insapp.
